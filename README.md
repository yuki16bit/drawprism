#### Front-End Tech Stack
- React.js (Hooks)
- React Router
- Redux Toolkit / RTK Query
- Socket.IO Client
- Html Canvas API
- Tailwind CSS

#### Backend Tech Stack
- Docker / docker-compose
- gunicorn
- gevent / gevent-websocket
- Python / Flask
- Flask-SocketIO
- Celery
- MongoDB Atlas
- AWS ElastiCache for Redis
- AWS S3
- AWS CloudFront
- AWS EC2
- JSON Web Token (JWT)
- MVC Design Pattern
- Ubuntu 20.04 (Windows Subsystem for Linux 2)

#### Net-working
- Nginx (Reverse Proxy / Http & Https support)
